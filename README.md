### 🚀 Featured Projects

#### 👁️ [Visual-Odometry-from-scratch](https://github.com/G4CiO/Visual_Odometry_from_Scratch)
Developed a real-time **Visual Odometry (VO)** system using **ROS2 Humble** and **OpenCV**. 
- Implemented stereo depth calculation (SGBM) and ORB feature tracking.
- Achieved a 5.13% trajectory error (0.45m RMSE) in Gazebo simulation.
- Optimized for real-time performance at a 12 Hz update rate.

#### 🚗 [Deep-Reinforcement-Learning-Mobile-Robot](https://github.com/G4CiO/Deep-Reinforcement-Learning-Mobile-Robot)
An **Automatic Parking Model** based on **Deep Reinforcement Learning (DDPG)**.
- Built a custom 2D kinematic car environment using Python.
- Engineered a reward function for precise parking within 10cm from the target.
- Trained over 50,000 timesteps to ensure convergence and safety.

#### 🤖 [Mobile-Robot-Lab (EKF Localization)](https://github.com/G4CiO/Mobile-Robot-LAB-1)
Implementation of **EKF Localization** and path-tracking controllers.
- Benchmarked PID, Pure Pursuit, and **Stanley Controllers** for high-precision path following.
- Developed a Sensor Fusion pipeline to fuse GPS and wheel odometry using an **Extended Kalman Filter**.
