### 🚀 Featured Projects

#### 👁️ [Visual-Odometry-from-scratch](https://github.com/G4CiO/Visual_Odometry_from_Scratch)

- Achieved a 5.13% trajectory error (0.45m RMSE) in Gazebo simulation.

- Engineered a reward function for precise parking within 10cm from the target.
- Trained over 50,000 timesteps to ensure convergence and safety.
